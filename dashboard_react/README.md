# Firewall Dashboard

A modern web application for monitoring and managing firewall rules, built with React, TypeScript, and Tailwind CSS.

## Features

- Real-time firewall monitoring
- Threat detection and analysis
- Traffic analytics and visualization
- Security integrations management
- AI-powered security agents
- Chat and collaboration tools
- Incident response management
- Data collection and analysis

## Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

## Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd firewall-dashboard
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will be available at `http://localhost:3000`.
